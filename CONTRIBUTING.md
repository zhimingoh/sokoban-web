# Contributing

## Quick Start

```bash
npm install
npm run dev
```

## Local Validation

Run all checks before pushing:

```bash
npm run lint
npm run typecheck
npm run build
```

## Branches

- Feature: `feat/<topic>`
- Fix: `fix/<topic>`
- Chore: `chore/<topic>`

## Pull Request Checklist

- [ ] Scope is clear and minimal
- [ ] Lint/typecheck/build pass locally
- [ ] UI manually verified
- [ ] Screenshots/GIF added for visible UI changes
- [ ] No secrets or tokens committed

## Commit Message Style

Examples:

- `feat: add level selector pagination`
- `fix: prevent box push through wall`
- `chore: align CI checks with local scripts`
